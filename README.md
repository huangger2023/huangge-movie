# 影视解说落地页 - 部署指南

## 📁 文件说明

```
my-landing-page/
├── index.html    # 主页面（需要修改个人信息）
├── style.css     # 样式文件（一般不用改）
├── qrcode.png    # 你的企业微信二维码（需要替换）
└── README.md     # 本说明文件
```

---

## 🔧 第一步：修改个人信息

打开 `index.html` 文件，修改以下内容：

### 1. 修改标题和名称
找到这两行，把 `【你的名字】` 替换成你的名字：
```html
<title>【你的名字】影视解说 - 专业教学</title>
<h1>【你的名字】影视解说</h1>
```

### 2. 修改微信号
找到这行，替换成你的微信号：
```html
<p>📱 如二维码失效，请添加微信：<strong>你的微信号</strong></p>
```

### 3. 修改口令（可选）
找到配置区域，修改口令：
```javascript
const CONFIG = {
    passwords: ['学习', '666', 'xuexi'],  // 可以设置多个口令
    ...
};
```

---

## 🖼️ 第二步：替换二维码

1. 把你的企业微信二维码保存为 `qrcode.png`
2. 放到项目文件夹中，替换原来的文件
3. 建议二维码尺寸：300x300 像素以上

---

## 🚀 第三步：部署到 Vercel（免费）

### 方法一：通过 GitHub 部署（推荐）

1. **上传到 GitHub**
   - 登录 https://github.com
   - 点击右上角 `+` → `New repository`
   - 仓库名填：`my-landing-page`
   - 选择 `Public`（公开）
   - 点击 `Create repository`
   - 点击 `uploading an existing file`
   - 把文件夹里的所有文件拖进去
   - 点击 `Commit changes`

2. **连接 Vercel**
   - 打开 https://vercel.com
   - 点击 `Sign Up` → 选择 `Continue with GitHub`
   - 授权后，点击 `Add New...` → `Project`
   - 选择你刚创建的 `my-landing-page` 仓库
   - 点击 `Deploy`
   - 等待部署完成，会给你一个网址如：`https://my-landing-page-xxx.vercel.app`

3. **绑定自定义域名（可选）**
   - 在 Vercel 项目设置中可以绑定自己的域名

### 方法二：直接拖拽部署

1. 打开 https://vercel.com
2. 注册/登录
3. 把整个 `my-landing-page` 文件夹拖到 Vercel 页面
4. 自动部署，获得网址

## 📊 查看访问统计

### 方法一：浏览器控制台
1. 打开你的网站
2. 按 F12 打开开发者工具
3. 在 Console（控制台）输入：`showStats()`
4. 查看访问量和转化率

### 方法二：接入专业统计（推荐）
1. 注册 https://www.51.la （免费）
2. 添加网站，获取统计代码
3. 把代码粘贴到 `index.html` 的 `<head>` 中

## ❓ 常见问题

**Q: 二维码过期了怎么办？**
A: 重新生成二维码，替换 qrcode.png 文件，重新上传到 GitHub，Vercel 会自动更新。

**Q: 想换口令怎么办？**
A: 修改 index.html 中的 passwords 数组，重新上传即可。

**Q: 能看到是谁访问的吗？**
A: 基础版只能看访问量，如需详细数据建议接入 51.la 统计。
